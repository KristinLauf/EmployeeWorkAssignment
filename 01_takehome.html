Take-home — Employee pay data

You are provided with two JSON files containing fictional employee records for January and February. Each file is an array where each element is a single employee record with the following fields:

  • employee_id
  • gender (1 = male, 2 = female)
  • job_role
  • experience_months
  • performance
  • has_mgmt_responsibility
  • education
  • salary
  • bonus

You’re welcome to use AI assistance — we assume you will. We care about the judgment behind your choices as much as the code itself, so be ready to explain and defend them. And where the brief doesn’t give you enough to go on, don’t get stuck — make a reasonable assumption, note it in your README, and keep going.

Using only the standard library of your chosen language, write a method for each task below. Aim for solutions that are correct, efficient, and clear and easy to read.

Submit your code (with tests) and a short README covering how to run it, the assumptions you made, and any choices worth explaining.

Tasks

  1. Generate a list of all job roles.
  2. Calculate the average salary for employees in the "Helpers--Roofers" job role.
  3. Which employees have left and which have joined between January and February?
  4. Who got the biggest raise between January and February?
  5. Given a target salary, find the employee whose salary is nearest to it.
    • What is the time complexity of your solution, and how would it change if you had to answer this for many target salaries, not just one?
  6. Given an employee, find the one most similar to them.

Please treat this document as confidential and don’t share it.
